public byte[] publish (String topicName, Long waitForCompletionTimeOut, MqttConnector.DeliveryQoS qos, byte[] messagePayload, MuleEvent muleEvent)

Publish a message to a topic. If sucessful a flow variable named MQTT_DELIVERY_TOKEN_VARIABLE will contain the MqttDeliveryToken that can be used for further awaiting completion.

Parameters
topicName Topic to publish message to.
waitForCompletionTimeOut Time in milliseconds to wait for the delivery to occur.
qos QoS level to use when publishing message.
messagePayload The payload that will be published over MQTT.
muleEvent The in-flight MuleEvent.
Returns
  • the byte[] that was published.
Throws
MqttException thrown if the MQTT publish fails.
